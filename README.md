# hardware-monitor-android
A program for measuring hardware parameters on Android smartphones. \
Displaying overlaid labels on the screen to stream hardware information, it has an app service to do that. \
The energy consumption during measurement is very low, it only measures when the screen is on.
The service will pause when the screen is off, and it will automatically resume when the screen turns back on.
